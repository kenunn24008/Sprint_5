# Open Notes

Simple note taking app that allows users to store notes to a personal account in a database.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Start a virtual device/Connect an andriod device to android studio.
2. Run and install the app

Instructions for using the software:

1. Create an account
2. Add a note title and text
3. Click save

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Andriod Studio
